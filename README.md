# Breast Cancer Detection using Machine Learning

This project implements a diagnostic tool to classify breast cancer tumors as **Malignant** or **Benign**. Using the classic Wisconsin Breast Cancer Dataset, the project compares two powerful algorithms: **Random Forest** and **Support Vector Machine (SVM)**.

## 📌 Project Overview
The goal of this study is to automate the diagnostic process by analyzing 30 clinical features (such as texture, radius, and area) extracted from digitized images of fine needle aspirate (FNA) of breast masses.

## 🛠️ Technologies Used
* **Python 3.12**
* **Scikit-Learn**: For model training, splitting, and metrics.
* **Matplotlib & Seaborn**: For data visualization and confusion matrices.
* **NumPy**: For numerical data handling.
